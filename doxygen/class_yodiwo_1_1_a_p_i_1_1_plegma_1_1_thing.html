string Yodiwo.API.Plegma.Thing.BlockType

Specifies the Thing's block type if it's to be specially modeled in the Cyan UI It can be left null if this Thing is to be modeled by the default Cyan UI blocks In this case Output-type Ports are gathered and represented as a Cyan UI Input Thing (thing->cloud events) and Input-type Ports are gathered and represented as a Cyan UI Output Thing (cloud->thing events) Both event directions occur via the PortEventMsg and PortEventBatchMsg messages
