Connect an Amazon edge node with Yodiwo Cloud Platform

Amazon Web Services

Section A: Setup AWS IoT Application

1.) Sign in to the AWS Management Console and open the AWS IoT console at https://aws.amazon.com/iot. On the Welcome page, choose Get started with AWS IoT.
2.) Choose the Create a resource button
3.) Choose Create a thing, type a name for the thing, and then choose Create
4.) Choose View thing to display information about your thing (note the rest api endpoint, the hostname of the mqtt broker id the substring after "https://" until ".com")
5.) Choose the Connect a device button to download a key pair (public private) and a certificate generated by AWS IoT
6.) Download a root CA certificate from here: https://www.symantec.com/content/en/us/enterprise/verisign/roots/VeriSign-Class%203-Public-Primary-Certification-Authority-G5.pem
6.) On the Connect a device page, select the SDK to use, and then choose Generate certificate and policy
